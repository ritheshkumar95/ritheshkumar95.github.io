# Analytics

This site is wired for GoatCounter, a free hosted analytics service for reasonable public usage.

Configured endpoint:

```text
https://ritheshkumar95.goatcounter.com/count
```

Dashboard after signup:

```text
https://ritheshkumar95.goatcounter.com/
```

Tracked data:

- Page visits on `/` and `/obamanet/`
- Referrers and campaign query parameters such as `utm_source`, `utm_medium`, and `utm_campaign`
- Viewer location/country as reported by GoatCounter
- Link clicks as events named `click:<source-page>:<kind>:<destination>`

The account name must be `ritheshkumar95` for the deployed endpoint to collect data without another code change.
